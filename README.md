# codex-trial

## Purpose
I want to use codex to develop a demo application in current repo to show the capability of codex.  

## Requirement
1. A backend micro service which provide restful API to manage students, including create, update, delete and query; 
2. The micro service is done using golang and advanced web framework as you know; 
3. The micro service is deployed using Kubernetes deployment; 
4. The CICD pipeline is built on gitlab;

## API Documentation

The full REST contract for the student service, including payload schemas, sample requests, and response codes, is documented in `docs/api.md`. Refer to that document whenever you need to integrate with or extend the HTTP interface.
